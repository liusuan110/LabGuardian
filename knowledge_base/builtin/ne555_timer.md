# NE555 定时器芯片 技术手册摘要

## 基本信息
- **型号**: NE555 / LM555 / SA555
- **类型**: 通用双极型定时器 IC
- **封装**: DIP-8 (最常见), SOP-8
- **供电**: 4.5V ~ 16V
- **用途**: 单稳态/双稳态触发器, 方波振荡, PWM 生成, 延时电路

## 引脚定义 (DIP-8, 顶视图)
```
       ┌──U──┐
  GND ─┤1   8├─ VCC
 TRIG ─┤2   7├─ DISCH
  OUT ─┤3   6├─ THRES
 RESET─┤4   5├─ CTRL
       └─────┘
```
- Pin 1 (GND): 接地
- Pin 2 (TRIG): 触发输入, 低于 VCC/3 时触发
- Pin 3 (OUT): 输出, 可灌/拉电流约 200mA
- Pin 4 (RESET): 复位, 低电平有效, 不用时接 VCC
- Pin 5 (CTRL): 控制电压, 可外接电压改变阈值, 不用时接 0.01μF 到 GND
- Pin 6 (THRES): 阈值输入, 高于 2×VCC/3 时复位
- Pin 7 (DISCH): 放电端, 内部接 NPN 集电极
- Pin 8 (VCC): 正电源

## 典型应用电路

### 无稳态振荡器 (方波发生器)
最常用配置, 产生连续方波:
- 频率: f = 1.44 / ((R_A + 2×R_B) × C)
- 占空比: D = (R_A + R_B) / (R_A + 2×R_B)
- 典型值: R_A=1kΩ, R_B=10kΩ, C=10μF → f ≈ 6.8Hz

接线方法:
1. Pin 8 (VCC) 接正电源, Pin 1 (GND) 接地
2. Pin 4 (RESET) 接 VCC (不复位)
3. R_A 从 VCC → Pin 7 (DISCH)
4. R_B 从 Pin 7 → Pin 2 (TRIG) = Pin 6 (THRES) (短接Pin2和Pin6)
5. C 从 Pin 2/6 → GND
6. Pin 3 (OUT) 为方波输出
7. Pin 5 (CTRL) 通过 0.01μF 接 GND

### 单稳态触发器 (延时)
负脉冲触发后, 输出高电平持续一段时间:
- 延时: t = 1.1 × R × C
- 示例: R=100kΩ, C=10μF → t ≈ 1.1秒

### 面包板注意事项
- 555 发热正常 (静态电流约 3-10mA)
- Pin 5 的去耦电容很重要, 可提高频率稳定性
- 大功率负载不要直接接 Pin 3, 通过三极管驱动
- 555 输出可以直接驱动 LED (注意串联限流电阻)

## 常见问题
Q: 555 不振荡怎么办?
A: 检查: 1) Pin 4 是否接 VCC; 2) Pin 2 和 Pin 6 是否短接; 3) 电容是否有极性接反; 4) 电源是否稳定。

Q: 555 输出频率不准?
A: 1) 电阻/电容实际值与标称值有偏差; 2) Pin 5 未接去耦电容; 3) 电源波纹大; 4) 面包板接触不良。
