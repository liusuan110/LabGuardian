# 常用芯片引脚速查与面包板接线指南

## 三极管引脚定义

### TO-92 封装 (8050, 8550, 2N3904, 2N3906, C945, 9013, 9014 等)
平面朝向自己, 引脚朝下:
- **左**: 发射极 (E)
- **中**: 基极 (B)
- **右**: 集电极 (C)

注意: 不同厂家可能引脚定义不同! 使用前务必查对应 datasheet!
特别注意: 2N3904/2N3906 的引脚顺序是 E-B-C, 但 BC547/BC557 是 C-B-E (与上面相反!)

### 常见 NPN / PNP 互补对
| NPN | PNP | 特点 |
|-----|-----|------|
| SS8050 | SS8550 | 大电流 (1.5A), 通用 |
| 2N3904 | 2N3906 | 经典小信号 |
| BC547 | BC557 | 欧系, 注意引脚不同! |
| 9013 | 9012 | 中电流 |
| 9014 | 9015 | 小信号, 高增益 |

## 运放芯片引脚

### LM741 (单运放, DIP-8)
```
       ┌──U──┐
 OFF1 ─┤1   8├─ NC
   V- ─┤2   7├─ V+supply
   V+ ─┤3   6├─ Vout
 V-sup─┤4   5├─ OFF2
       └─────┘
```

### LM358 / LM324 (双运放 / 四运放)
LM358 (DIP-8):
```
       ┌──U──┐
 Out1 ─┤1   8├─ VCC
  1(-) ─┤2   7├─ Out2
  1(+) ─┤3   6├─ 2(-)
  GND  ─┤4   5├─ 2(+)
       └─────┘
```

### NE5532 (双运放, DIP-8)
```
       ┌──U──┐
 Out1 ─┤1   8├─ VCC
  1(-) ─┤2   7├─ Out2
  1(+) ─┤3   6├─ 2(-)
  VEE  ─┤4   5├─ 2(+)
       └─────┘
```

## 面包板通用接线规范

### 电源接线
1. 红色电源轨 → VCC (正电源)
2. 蓝色电源轨 → GND (地)
3. 双电源场合: 上红=+VCC, 下蓝=-VEE, 中间某行作为 GND 参考
4. 检查: 全尺寸面包板电源轨中间可能断开!

### IC 芯片接线
1. 芯片跨骑中间沟槽, 确认 Pin 1 方向 (缺口/圆点)
2. VCC 和 GND 先接
3. 就近放去耦电容 (0.1μF)
4. 再接信号线

### 元器件方向
- 电阻: 无极性, 随意方向
- 陶瓷电容: 无极性
- 电解电容: 有极性! 长脚正极
- LED: 有极性! 长脚阳极 (+)
- 二极管: 有银环端为阴极 (-)
- 三极管: 注意 E/B/C 引脚顺序

### 调试技巧
1. 分段调试: 先电源→再偏置→再信号
2. 用万用表检查关键节点电压
3. 示波器看波形前确认探头衰减比
4. 接地夹接在电路地上, 不要浮空
5. 如果电路不工作, 先检查最简单的: 电源、地、芯片方向

## 电路常见故障排查清单
1. ☐ 电源电压正确? (用万用表测电源轨)
2. ☐ 地线连通? (面包板两侧 GND 轨是否连接)
3. ☐ IC 方向正确? (Pin 1 标记)
4. ☐ 有极性元件方向正确? (LED/电解电容/二极管/三极管)
5. ☐ 跳线是否接触良好? (试着按压或更换)
6. ☐ 相邻行有无意外短路?
7. ☐ 电阻/电容值是否正确? (色环/标识确认)
8. ☐ 芯片是否损坏? (换一个试试)
9. ☐ 示波器/万用表设置正确?
10. ☐ 焊接/插接是否有虚焊/虚插?
