# 模拟电路实验指南

## 面包板基础

### 面包板内部连接规则
面包板 (Breadboard / 无焊面包板) 的内部导电条连接方式:
- **主区域 (a-e 和 f-j)**: 每一行 (Row) 的 5 个孔 (a-e 或 f-j) 内部相连
- **中间沟槽**: a-e 和 f-j 之间有一条隔断沟, 不导通 (专为 DIP IC 设计)
- **电源轨**: 面包板上下两侧的长条, 同色同一条轨内部全部相连
  - 红色轨: 通常接 VCC (正电源)
  - 蓝色轨: 通常接 GND (地)
- **注意**: 有些全尺寸面包板 (830孔) 的电源轨在中间断开, 需要跳线连接!

### DIP 芯片插法
DIP (双列直插) 芯片跨骑中间沟槽插入:
- 芯片左半部分占 a-e 列
- 芯片右半部分占 f-j 列
- 每个引脚独占一行, 不会和对面引脚短路
- 插入前注意芯片方向: 缺口/圆点标记在上方 (Pin 1 端)

### 常用连线技巧
1. 电源和地线先接好并检查
2. 芯片先插好再连周边元件
3. 跳线颜色约定: 红=VCC, 黑/蓝=GND, 其他颜色=信号
4. 尽量短线连接, 减少引线电感/串扰
5. 高频电路在 IC 电源脚就近放 0.1μF 去耦电容

## 常见模拟电路实验

### 实验 1: 共射放大电路 (三极管)

#### 目的
- 理解三极管共射极放大原理
- 学习静态工作点 (Q点) 的设置与调整
- 测量电压增益, 输入/输出电阻

#### 典型电路参数 (以 8050 NPN 为例)
- VCC = 12V
- R_C (集电极电阻) = 3kΩ
- R_B (基极偏置电阻) = 300kΩ (固定偏置) 或分压偏置 R_B1/R_B2
- R_E (发射极电阻) = 1kΩ (加旁路电容 C_E = 100μF 时增益更大)
- 输入耦合电容 C_1 = 10μF, 输出耦合电容 C_2 = 10μF
- 信号源: 1kHz 正弦波, 幅值 10-50mV

#### 静态工作点设置
1. 固定偏置: I_B = (VCC - V_BE) / R_B, I_C ≈ β × I_B
2. 分压偏置 (更稳定): V_B = VCC × R_B2/(R_B1+R_B2), I_E ≈ (V_B - 0.7V) / R_E
3. 调整 R_B 使 V_CE ≈ VCC/2 (留有充足的输出摆幅)

#### 测量方法
- 静态: 用万用表直流档测 V_B, V_C, V_E (断开信号源)
- 动态: 用示波器同时观察输入和输出波形, 注意反相关系
- 电压增益: A_v = V_out(pk-pk) / V_in(pk-pk)

#### 常见问题
- Q点设置不当: V_CE 太大→截止失真, V_CE 太小→饱和失真
- 忘记接耦合电容: 直流耦合会改变后级电路的工作点
- 三极管引脚接错: E/B/C 反了电路完全不工作

### 实验 2: 运放反相/同相放大器

#### 目的
- 验证运放的"虚短"和"虚断"
- 测量闭环增益, 与理论值对比
- 观察带宽和转换速率限制

#### 反相放大器
- 运放: LM741 或 LM358
- R_1 = 1kΩ (输入电阻)
- R_f = 10kΩ (反馈电阻)
- 理论增益: A_v = -R_f / R_1 = -10
- 输入: 1kHz, 100mV 正弦波
- 供电: ±12V (741) 或 +12V 单电源 (358)

#### 同相放大器
- R_1 = 1kΩ, R_f = 9kΩ
- 理论增益: A_v = 1 + R_f / R_1 = 10
- 同相: 输出与输入同相位

#### 测试要点
1. 先接电源, 再接信号 (防 latch-up)
2. 观察输出是否在线性范围内 (不饱和)
3. 逐渐增大输入幅度, 观察何时开始削波
4. 改变频率, 观察高频下增益下降 (带宽限制)

#### 常见问题
- 输出饱和: 输入太大, 导致输出超出电源范围
- 振荡: 反相端和同相端接反了, 变成正反馈
- 无输出: 检查电源是否连接、芯片是否损坏

### 实验 3: 有源滤波器

#### 一阶低通有源滤波器
- 在反相放大器的反馈电阻 R_f 上并联电容 C
- 截止频率: f_c = 1/(2π × R_f × C)
- 通带增益: |A_v| = R_f / R_1
- 高频衰减: -20dB/dec

#### 一阶高通有源滤波器
- 在输入电阻 R_1 前串联电容 C
- 截止频率: f_c = 1/(2π × R_1 × C)

#### 测量方法 (频率响应)
1. 输入信号保持幅度不变, 改变频率 (如 10Hz 到 100kHz)
2. 每个频率测量输出幅度
3. 画 Bode 图: 横轴 log(f), 纵轴 20×log(Vout/Vin) dB
4. 找到 -3dB 点即为截止频率

### 实验 4: 信号发生器 (方波/三角波)

#### 施密特触发器 + 积分器
1. 运放 1: 接成迟滞比较器 (施密特触发器) → 输出方波
2. 运放 2: 接成积分器, 输入来自方波 → 输出三角波
3. 调节积分器 RC 和比较器的阈值改变频率和幅度

## 实验安全须知

### 电气安全
1. **先断电再接线**, 接好后再通电
2. 检查电源极性是否正确, 特别是电解电容和极性元器件
3. 电源电压不超过器件额定值
4. 短路后立即断电! (面包板最容易发生相邻行短路)

### 元器件保护
1. LED 必须串联限流电阻 (通常 220Ω-1kΩ)
2. 电解电容注意极性 (长脚正极)
3. 三极管/MOSFET 注意引脚顺序
4. IC 芯片不要带电插拔
5. 运放供电后不要悬空输入端 (接地或给确定电平)

### 测量注意
1. 示波器探头衰减比设置正确 (1X / 10X)
2. 万用表量程选择合适, 从大量程开始
3. 测量前确认探头/表笔接触良好
4. 多通道示波器注意共地问题

## 常用元件参数速查

### 电阻色环读法 (四环)
| 颜色 | 数字 | 乘数 | 容差 |
|------|------|------|------|
| 黑 | 0 | ×1 | — |
| 棕 | 1 | ×10 | ±1% |
| 红 | 2 | ×100 | ±2% |
| 橙 | 3 | ×1k | — |
| 黄 | 4 | ×10k | — |
| 绿 | 5 | ×100k | ±0.5% |
| 蓝 | 6 | ×1M | ±0.25% |
| 紫 | 7 | ×10M | ±0.1% |
| 灰 | 8 | — | — |
| 白 | 9 | — | — |
| 金 | — | ×0.1 | ±5% |
| 银 | — | ×0.01 | ±10% |

示例: 棕-黑-红-金 = 10 × 100 = 1kΩ ±5%

### 常用电阻值 (E12系列)
1Ω, 1.2Ω, 1.5Ω, 1.8Ω, 2.2Ω, 2.7Ω, 3.3Ω, 3.9Ω, 4.7Ω, 5.6Ω, 6.8Ω, 8.2Ω
及其 10 的整数次幂倍 (如 100Ω, 1kΩ, 10kΩ...)

### 电容标识
- 陶瓷电容: 数字标识, 如 "104" = 10 × 10^4 pF = 100nF = 0.1μF
- 电解电容: 直接标注容量和耐压, 如 "100μF 25V"
- 极性: 电解电容有极性! 长脚正极, 短脚负极 (或标有负极标记)

### LED 正负极识别
- 长脚 = 阳极 (+, Anode)
- 短脚 = 阴极 (-, Cathode)
- 内部较大的金属片 = 阴极
- 外壳有平切面的一侧 = 阴极
- 正向导通压降: 红色 ~1.8V, 黄色 ~2.0V, 绿色 ~2.2V, 蓝色 ~3.0V, 白色 ~3.2V
