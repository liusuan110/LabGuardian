# SS8550 PNP 三极管 技术手册摘要

## 基本信息
- **型号**: SS8550 (也称 S8550)
- **类型**: PNP 硅三极管
- **封装**: TO-92 (常见), SOT-23 (贴片)
- **制造商**: 多家 (长电科技, UTC, ON Semi 等)
- **用途**: 低频放大, 开关电路, 与 SS8050 互补配对

## 引脚定义 (TO-92 封装)
正面朝向自己 (有字面), 引脚朝下:
- 左: **发射极 (E, Emitter)**
- 中: **基极 (B, Base)**
- 右: **集电极 (C, Collector)**

与 8050 引脚排列完全一致！只是电流方向相反。

## PNP 三极管工作原理
与 NPN 的关键区别:
- **电流方向**: 电流从发射极(E) → 集电极(C), 基极电流从 E → B
- **导通条件**: V_BE < -0.7V (即基极电位低于发射极至少 0.7V)
- **偏置方式**: 发射极接 VCC (高电位), 集电极接负载 → GND

简易记忆:
- NPN: 基极给高电平导通 → "高开"
- PNP: 基极给低电平导通 → "低开"

## 面包板使用指导

### 基本共射放大电路 (PNP)
注意: PNP 管电路与 NPN "上下颠倒"
1. 发射极(E) 接 VCC (正电源)
2. 基极(B) 通过电阻接控制信号
3. 集电极(C) 通过负载电阻接 GND
4. 基极需要拉低才能导通!

### 典型应用: 高端开关
PNP 管最常见用途是"高端开关" (High-Side Switch):
- VCC → 8550 发射极(E)
- 8550 集电极(C) → 负载 → GND
- MCU GPIO 通过限流电阻 (10kΩ) → 8550 基极(B)
- GPIO 输出低电平 = 负载通电; 高电平 = 负载断电

### 与 8050 (NPN) 配对使用

#### 推挽输出 (互补对管)
常用于音频功放输出级:
```
              VCC
               |
          E ── 8550 (PNP) ── C ──┐
                                  ├── 输出 (Speaker)
          C ── 8050 (NPN) ── E ──┘
               |
              GND
```
- 正半周: 8550 导通, 电流从 VCC → 8550 → 负载
- 负半周: 8050 导通, 电流从负载 → 8050 → GND
- 效率远高于单管放大

#### H 桥电机驱动
使用 2 个 8050 + 2 个 8550 组成 H 桥:
```
     VCC            VCC
      |              |
   E─8550─C      C─8550─E
      |      Motor    |
   C─8050─E      E─8050─C
      |              |
     GND            GND
```
- 左 8550 + 右 8050 导通 → 电机正转
- 右 8550 + 左 8050 导通 → 电机反转
- 注意: 同侧两管不能同时导通 (短路!)

### 注意事项
- PNP 管发射极一定接高电位 (VCC), 集电极接低电位方向
- 基极电压要低于发射极才能导通, 这和 NPN 相反
- 不要把 PNP 和 NPN 的电路原封不动互换, 电源方向都要反转
- TO-92 封装引脚顺序 E-B-C 与 8050 相同, 但注意内部结构是反的

## 常见问题
Q: 如何区分 8050 和 8550 外观？
A: 看三极管表面丝印! "8050" = NPN, "8550" = PNP。两者TO-92封装外观完全相同, 必须看型号标识。

Q: 为什么实验中 PNP 管总是无法正常工作？
A: 最常见错误: 把 PNP 当 NPN 接了。PNP 的发射极必须接 VCC (高电位), 基极需要拉低导通, 这和 NPN 完全相反。

Q: 8050 和 8550 可以互相替换吗？
A: 不可以直接替换! 它们是互补对管, 极性相反。如果电路设计用 NPN (8050), 换成 PNP (8550) 需要把整个偏置电路反转。

Q: 怎么用万用表测量 8550？
A: 二极管档: E→B 正向约 0.6-0.7V, C→B 正向约 0.6-0.7V (注意方向和 NPN 反), 其他方向应该无穷大。如果使用 NPN 的测法, 反过来即可。
